
# CSN Exam Registration Website

CIT_260_CSN_Exam_Website was originally a webpage done as a class project by me and three other classmates.

The original version of the website used Python Flask and other backend technologies to run the backend. Since GitHub Pages only hosts static websites, I created a fork of the project and redid the backend functionality using JavaScript so the website could be displayed as a static site.

The purpose of the website is to allow students to view and register for exams. It also allows faculty members to review exam registrations and print reports showing which students have registered for exams.

The aesthetic design of the webpage was indented to resemble the design of the https://www.csn.edu/ website page.

https://javicshz.github.io/CIT_260_CSN_Exam_Website/

## Files

The main folder contains two folders:

-   `/Docs/`  
    Contains the static redone version of the website displayed through GitHub Pages.
-   `/Original Documents/`  
    Contains the original unaltered version of the website.

## Account Logins

For ease of use and privacy reasons, the original user logins have been removed and replaced with generic stand-in accounts.

When on the login page, the webpage will display generic user logins that can be used for testing purposes.
